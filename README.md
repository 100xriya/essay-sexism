# Sexism in Indian Schools and Strict Gendered Dress Codes

A simple HTML5 webpage discussing sexism in Indian schools and the impact of strict gendered dress codes on students.

## Features

* Semantic HTML structure
* Headings, paragraphs, lists, and links
* Text formatting elements
* Interactive `<details>` section
* Basic HTML button

## Technologies Used

* HTML5

## Purpose

This project was created to practice HTML fundamentals while presenting information on gender inequality in educational institutions.
